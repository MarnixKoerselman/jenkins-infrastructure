Some notes to configure a VM (or bare metal) host that runs Ubuntu:

Change the network configuration:
https://linuxconfig.org/how-to-configure-static-ip-address-on-ubuntu-18-04-bionic-beaver-linux
https://www.tecmint.com/configure-network-static-ip-address-in-ubuntu/

Install docker:
https://docs.docker.com/install/linux/docker-ce/ubuntu/

e.g.:
    $ curl -fsSL https://get.docker.com -o get-docker.sh
    $ sudo sh get-docker.sh
    $ sudo usermod -aG docker your-user

To configure Jenkins to use a different Docker host, create Jenkins credentials of type:
    Docker Host Certificate Authentication
Use the contents of generated certs and key to fill the entries in Jenkins > Configure System > Cloud > Docker

By default, on Ubuntu, it isn't possible to specify hosts in daemon.json. Use "sudo systemctl edit docker.service" to specify an override
(see https://support.openanalytics.eu/t/how-to-configure-docker-startup-on-ubuntu-18/842/2 )
Copy Docker server certs to /var/docker/
Copy Docker client certs to ~/.docker/
